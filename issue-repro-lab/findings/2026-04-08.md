## 이슈별 원인

### #4 Prompt / Output Monitoring
- 재연 방식:
  - task 에서 `ctx.log.info(..., { prompt, promptLength })`
  - `ctx.ai.chat()` 호출
  - `ctx.log.success(..., { outputLength })`
  - line renderer 출력 확인
- 관찰 결과:
  - renderer 출력에는 message 만 보이고 `prompt`, `promptLength`, `outputLength` 가 보이지 않는다.
- 원인:
  - `ctx.ai.chat()` / `ctx.ai.chatStream()` 경로에 자동 prompt/output monitoring 이 없다.
  - line renderer 가 `TaskLogEvent.meta` 를 화면에 표시하지 않는다.

### #14 New Chat Input Shape
- 재연 방식:
  - `AIChatRequest.messages[].content` 에 `{ type, text }[]` 를 넣는 fixture 를 TypeScript program 으로 컴파일
- 관찰 결과:
  - `Type '{ type: string; text: string; }[]' is not assignable to type 'string'.`
- 원인:
  - `AISessionMessage.content` 와 provider 변환 경로가 string 기반으로 고정돼 있다.

### #15 Memory Write Drop
- 재연 방식:
  - large workflow raw memory record 를 반복 저장
  - `write()` disposition 과 `process.memoryUsage().heapUsed` 를 함께 기록
- 관찰 결과:
  - 첫 write 는 `create`
  - 이후 유사 record 는 예외 없이 `discard`
  - heap 사용량은 증가하지만 write 실패는 heap exhaustion 이 아니라 heuristic discard 다
- 원인:
  - novelty 감소로 importance 가 `0.6` 아래로 내려가고, `write()` 가 조용히 `discard` 를 반환한다.
  - 호출자가 disposition 을 확인하지 않으면 "정상 저장" 처럼 보인다.

### #16 Ink Workflow Chart Infinite Stack
- 재연 방식:
  - workflow branch 가 자기 자신을 child workflow 로 가리키는 cyclic state 를 `InkRendererScreen` 에 주입
- 관찰 결과:
  - `Maximum call stack size exceeded`
  - `renderWorkflowBranchLines` stack 이 반복된다.
- 원인:
  - child workflow recursion 에 visited guard 가 없다.

### #17 Missing Error Stack Trace
- 재연 방식:
  - task 가 stack trace 를 가진 `Error` 를 throw
  - line renderer 출력과 `result.error.details.stack` 비교
- 관찰 결과:
  - stack trace 는 result object 에 남아 있다.
  - renderer 출력에는 failure summary 만 보이고 stack 이 사라진다.
- 원인:
  - task failure renderer 가 `errorDetails.stack` 을 출력하지 않는다.

### #18 Log Color Theme Missing
- 재연 방식:
  - `line` renderer 를 `useColor: true` 로 생성
  - `colorConfigPath` 에 warn 색상을 `magenta` 로 지정
  - `warn` level task log 출력 확인
- 관찰 결과:
  - 기존 구현은 ANSI 색상을 적용하지 않고, config file 에서 color mapping 을 읽는 경로도 없다.
- 원인:
  - renderer 에 level/theme abstraction 이 없고, line/compact/Ink 출력이 모두 plain text 로만 고정돼 있다.
