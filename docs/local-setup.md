# Local Setup

## Requirements
- Node.js 22+
- npm 10+

## Install
```bash
npm install
```

## Useful commands
```bash
npm run typecheck
npm test
npm run build
npm run example
```

## CLI rendering
- live rendering on:
```bash
npm run example
```
- line mode:
```bash
CYCLE_RENDER_MODE=line npm run example
```
- live rendering off:
```bash
CYCLE_LIVE=0 npm run example
```
